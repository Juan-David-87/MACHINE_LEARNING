What is Logistic Regression?

Logistic Regression is a supervised machine learning algorithm used for classification problems. Instead of predicting a continuous value, it predicts the probability that an observation belongs to a certain category.

The output is a value between 0 and 1, which represents probability.

Regression vs Classification

Regression predicts continuous values (e.g., oil quantity, price).

Classification predicts categories or classes (e.g., High vs Low imports).

Binary Classification

Logistic Regression is commonly used for binary classification.

  • 0 → Negative / Low
  • 1 → Positive / High

Target Variable

The target variable is what we want to predict. In logistic regression, it is usually categorical (0 or 1).

Predictor Variables

Predictor variables are the inputs used to make predictions (e.g., year, demand, price).

Probability

Logistic Regression outputs a probability between 0 and 1.

  • 0.2 → Low probability of High Imports
  • 0.8 → High probability of High Imports

Decision Boundary

The decision boundary separates one class from another based on probability.

Classification Threshold

The typical threshold is 0.5:

  • If probability ≥ 0.5 → Class 1 (High)
  • If probability < 0.5 → Class 0 (Low)
